# CardioTwin AI Model Metrics Summary

This report is generated by `notebooks/08_model_comparison_report.ipynb` after the MLflow training notebooks run.

Expected outputs:

- Best cardiovascular status classification model by weighted F1 score.
- Best blood pressure regression model by RMSE.
- Accuracy, precision, recall, F1 score, MAE, RMSE, and R2 values from MLflow.
- Short explanation of why each best model was selected.

The bundled dataset is synthetic and intentionally small, so initial metric values are only for workflow validation.
