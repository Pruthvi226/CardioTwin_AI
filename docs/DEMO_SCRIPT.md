# Demo Script

CardioTwin AI is an end-to-end AI platform for wearable PPG signal analytics. It takes raw signal input, cleans it, extracts statistical and frequency-domain features, converts the signal into spectrograms for a Computer Vision branch, and runs predictive models using Scikit-learn and PyTorch. The dashboard shows preprocessing, prediction, confidence, uncertainty, explainability, and automated experiment reporting. I also added FastAPI inference and experiment tracking to make it closer to a real AI engineering workflow.

Research demo only. Not for clinical diagnosis or medical decision-making.
