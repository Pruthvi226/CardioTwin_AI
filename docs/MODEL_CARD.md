# Model Card

## Model Purpose
Classify stress-like or signal-quality patterns and estimate continuous demo scores from PPG windows.

## Intended Use
AI internship demo, signal analytics research, and pipeline validation.

## Not Intended Use
Clinical diagnosis, medical decision-making, triage, treatment planning, or patient recommendations.

## Dataset
Default results use synthetic demo data. Real mode is documented but requires a governed external dataset.

## Evaluation Metrics
Accuracy, precision, recall, weighted F1, MAE, RMSE, R2.

## Limitations
Synthetic validation does not prove real-world generalization. Subject variability and sensor noise can change performance.

## Ethical Considerations
Research demo only. Not for clinical diagnosis or medical decision-making.
